Model Architecture

Four custom components working together — dual-stream encoding, gated fusion, and diffusion-based generation.

Temporal Encoder

154,752 params · 4 layers

Multi-head self-attention over the agent's own trajectory history. Captures velocity profiles, acceleration patterns, and motion dynamics using positional encoding + learned CLS token.

Social Encoder

109,376 params · 3 layers

Cross-attention where the ego agent attends to neighbors. Uses relative position, velocity, distance, and bearing features to model collision avoidance and group behavior.

Gated Fusion

16,832 params

Sigmoid-gated blending of temporal and social context. The gate learns when to weight individual motion vs social interaction signals for each prediction.

Diffusion Decoder

389,058 params · 6 layers

Conditional DDPM with cosine noise schedule and AdaLN-modulated Transformer denoiser. Supports both full DDPM (100 steps) and DDIM (5–50 steps) sampling for diverse trajectories.

Diffusion Denoising

Watch noise become motion. The reverse diffusion process progressively refines random Gaussian noise into coherent trajectory predictions.

Run It Yourself

cd MotionTransformer
pip install -r requirements.txt

# run full demo (train → evaluate → visualize)
python demo/quick_demo.py

# or run unit tests
python tests/test_model.py
